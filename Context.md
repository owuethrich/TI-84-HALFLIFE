# TI-84 "HALFLIFE"
## Purpose
The program is formatted around the principles of nuclear decay. By using an exponential function with base 0.5, we can model atomic decay with a TI-84 program.

## Function
### Inputs
+ P(final) = Final mass after decay; measured in mass units (have to be the same units measured in P(original)).
+ P(original) = Original mass before decay; measured in mass units.
+ Time = Changed in time between the original and final mass; measured in time units (have to be the same units measured in Half Life).
+ Half Life = The half life of the atoms in the substance; how long before you have half of your original value; measured in time units.
+ 'e' - Used as a variable in the program to determine the desired value.
### Outputs
+ The final output is the desired/unknown value 'e'.
